# Checkbox

Accessible and themeable form checkbox input component.

```js
import { Label, Checkbox } from '@rebass/forms'
```

```jsx live
<Box>
  <Label>
    <Checkbox
      id='remember'
      name='remember'
    />
    Remember me
  </Label>
</Box>
```

Note that the `Label` component uses `display: flex` by default to make alignment with labels simpler.
