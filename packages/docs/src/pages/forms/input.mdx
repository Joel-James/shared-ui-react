# Input

Accessible and themeable form input component.

```js
import { Label, Input } from '@rebass/forms'
```

```jsx live
<Box>
  <Label htmlFor='email'>Email</Label>
  <Input
    id='email'
    name='email'
    type='email'
    placeholder='jane@example.com'
  />
</Box>
```
